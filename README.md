# 📊 Retail Sales Data Analysis (SQL Project)

## 📌 Project Overview
This project analyzes a Retail Sales dataset using PostgreSQL to extract meaningful business insights.  
The analysis includes data cleaning, data exploration, and solving business problems using SQL queries.

The goal of this project is to demonstrate SQL skills required for Data Analyst roles, including aggregations, filtering, window functions, and Common Table Expressions (CTEs).

---

## 🛠 Tools & Technologies
- SQL
- PostgreSQL
- pgAdmin

---

## 📂 Dataset Information
The dataset contains retail transaction records with the following fields:

- Transaction ID  
- Sale Date  
- Sale Time  
- Customer ID  
- Gender  
- Age  
- Product Category  
- Quantity  
- Price per Unit  
- Cost of Goods Sold (COGS)  
- Total Sale  

---

## 🔍 Data Cleaning
Steps performed during data preparation:

- Checked for NULL values
- Removed rows with missing quantity
- Updated missing age values
- Renamed incorrect column names
- Ensured data consistency

---

## 📊 Data Exploration
Basic exploration performed on the dataset:

- Total number of sales
- Total number of customers
- Unique product categories

---

## 📈 Business Questions Solved

1. Retrieve all sales made on a specific date  
2. Find transactions for Clothing category with quantity greater than 4 in Nov 2022  
3. Calculate total sales for each category  
4. Find the average age of customers purchasing Beauty products  
5. Identify transactions with total sales greater than 1000  
6. Count transactions by gender and category  
7. Identify the best selling month in each year using window functions  
8. Find top 5 customers based on total sales  
9. Count unique customers per category  
10. Analyze number of orders by time shifts (Morning, Afternoon, Evening)

---

- WINDOW FUNCTIONS (RANK)
